# 考研择校工作台

这是一个面向考研择校的静态网页工具，用来把研招院校、专业目录、初试科目和复试线参考集中到一个可筛选、可对比的页面里。

## 网站用途

考研择校时，信息通常分散在院校库、硕士专业目录、学校招生网和复试线公告中。这个网站的目标是把这些信息整理成一个更直观的工作台，帮助用户按专业、地区、院校层级、初试外语和预估分数快速缩小目标学校范围。

## 核心功能

- 按专业大类、一级学科和具体专业筛选招生单位
- 按省份、院校标签和主管部门查看学校分布
- 按初试外语科目筛选学校，例如英语、俄语、日语等
- 查看部分专业的近三年复试线参考
- 输入预估分数后，将学校分为冲刺、稳妥、保底和风险较高
- 设置目标学校，并生成每日备考任务
- 支持自定义每日任务和完成状态记录

## 数据说明

页面使用整理后的本地静态数据文件运行，不依赖后端服务器。专业目录和初试科目数据来自公开招生信息整理，分数线数据用于择校参考，具体报考仍应以学校研究生院和中国研究生招生信息网发布的最新信息为准。

## 适合人群

- 正在初步确定考研目标院校的学生
- 想按专业快速筛选学校的学生
- 想结合预估分数判断择校梯度的学生
- 需要把目标院校拆成日常复习任务的学生

## 使用提醒

本工具适合做择校前期筛选和备考规划参考，不替代官方招生简章、专业目录和复试录取办法。正式报考前，请再次核对目标学校当年的招生专业、考试科目、招生人数和复试线要求。
